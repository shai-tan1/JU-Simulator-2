JU Simulator 2
Hey! This is JU Simulator 2. It's a 3D university life sim where you run around campus and try to survive the week.

You get 5 days to manage your life, go to classes, and try not to fail out.

Play in your browser
You can play the web version right here: https://shai-tan1.github.io/JU-Simulator-2/
wait for 2 minutes, it takes time for the first time

What's the game about?
You basically just walk around and try to keep your stats in check. The UI on the top left shows how you're doing:

Attendance: Skip too many classes and you're in trouble.
Energy: Doing stuff takes energy. Don't pass out.
Mood: Gotta keep your sanity up.
CGPA: Starts at 7.50. Try to keep it decent.
Money: You start with ₹300. Spend it wisely.
Tech stuff
The game is built using the Godot Engine. The web version is exported to HTML5 and hosted right on GitHub Pages.

How to run it locally

Clone the repo.
Open project.godot in Godot 4.
Hit the play button.
If you just want to play the web build without opening Godot, go to the docs folder and open index.html in your browser (though sometimes Godot web builds need a local server to run properly, so using the GitHub Pages link above is easier).
